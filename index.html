Tips
17
Tips สำหรับผู้ใช้งาน
เทคนิคที่ช่วยให้ได้ผลลัพธ์ดีขึ้น
💬
ตอบคำถาม AI ให้ละเอียด
ยิ่งให้ข้อมูล context มากเท่าไหร่ AI ยิ่งสร้าง output ที่ตรงใจ ไม่ต้องกลัวพิมพ์ยาว
🔄
ทำทีละขั้นตอน
ตรวจสอบ output แต่ละขั้นก่อนไปขั้นถัดไป — แก้ไขได้ ไม่ต้องรอจนจบ
❓
ใช้ /bmad-help เสมอ
ไม่แน่ใจว่าจะเริ่มจากตรงไหน? พิมพ์ /bmad-help แล้วบอกสิ่งที่ต้องการ
📂
เปิดใน folder โปรเจกต์
ต้องเปิด Claude Code ใน folder ที่ติดตั้ง BMAD ไว้ ไม่งั้นจะไม่เห็นคำสั่ง
✏️
แก้ไข output ได้ตลอด
ไฟล์ output เป็น Markdown ธรรมดา — เปิดแก้ได้ หรือใช้ /bmad-bmm-edit-prd ให้ AI ช่วย
🎯
